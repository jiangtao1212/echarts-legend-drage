# vue-echarts-drag

This template should help get you started developing with Vue 3 in Vite.

此组件是基于 echarts + vue-draggable-plus 的图例拖拽组件，可以实现图例合并和分离功能。

**注：** 演示案例中拖拽时会有一些阴影块这是GIF截图软件（LICEcap）造成的，实际运行无此问题。
